# HTML Newsletter Portfolio Sample

This repository contains a responsive HTML email newsletter created as a portfolio sample.

The project is designed using email‑safe HTML and CSS patterns, focusing on real‑world
constraints such as limited CSS support, cross‑client compatibility, and responsive behavior.

## Live Preview
👉 https://jimenezjaik-web.github.io/html-newsletter-portfolio/

## Key Features
- Table‑based layout for broad email‑client support
- Responsive structure optimized for desktop and mobile
- Externally hosted images (email delivery best practice)
- Clear content hierarchy and call‑to‑action design
- Optimized typography for readability

## Technical Notes
- Built specifically for HTML email (not standard web layouts)
- CSS kept email‑safe and compatibility‑focused
- Designed to degrade gracefully if images are disabled

## Use Case
This sample represents a marketing or campaign newsletter and demonstrates
practical HTML email development skills suitable for production environments.

---

📌 **Note:** Rendering may vary slightly across email clients due to client‑specific CSS limitations.
